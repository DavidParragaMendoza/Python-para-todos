## 2. Los Operadores Numéricos en Python

Se definen los símbolos estándar para las operaciones:

- `+` **Suma** y `-` **Resta**: Funcionan como se espera.
- `*`**Multiplicación**: Se usa el asterisco.
- `/` **División**: Se usa la barra inclinada.
- `**` **Potencia:** Se usan dos asteriscos.
- **`%`** **Módulo (Residuo):** El profesor destaca este operador como muy útil en computación. No devuelve el resultado de la división, sino el **sobrante o residuo** de una división entera.

### 3. Orden de Evaluación (Precedencia de Operadores)

Python necesita reglas estrictas para saber qué operación realizar primero cuando se encuentra con una línea compleja. El profesor explica la jerarquía de poder, de mayor a menor importancia:

1. **Paréntesis `()`:** Son la regla suprema. Todo lo que esté dentro de paréntesis se evalúa primero. El profesor recomienda usarlos siempre para hacer el código legible.
2. **Potencia `*`:** Se evalúa antes que multiplicaciones o sumas.
3. **Multiplicación , División `/` y Módulo `%`:** Tienen el mismo nivel de jerarquía.
4. **Suma `+` y Resta `-`:** Son los últimos en evaluarse.
5. **Regla de Izquierda a Derecha:** Si los operadores tienen el mismo nivel, Python evalúa de izquierda a derecha.

### 4. Ejemplo Práctico

El video analiza paso a paso la expresión: 

$$
x = 1 + 2 ** 3 / 4 * 5.
$$

1. Primero la **Potencia**: 2^3 = 8.
    
    $$
    x = 1 + 8 / 4 * 5.
    $$
    
2. Luego **División**: 8 / 4 = 2.0
    
    $$
    x = 1 + 2.0 * 5.
    $$
    
3. Luego **Multiplicación**: 2.0 * 5 = 10.0.
    
    $$
    x = 1 + 10.0
    $$
    
4. Finalmente **Suma**: 1 + 10.0 = 11.0
    
    $$
    x =11.0
    $$
    

### 💡 Conclusión Clave

Aunque existen reglas estrictas de precedencia que Python sigue a la perfección, el consejo del Dr. Chuck para los programadores es: **Usa paréntesis**. 

```
x = 1 + (((2 ** 3) / 4) * 5).
```

No obligues a quien lea tu código (o a ti mismo en el futuro) a memorizar la tabla de precedencia; usa paréntesis para hacer tus intenciones matemáticas claras y explícitas.
