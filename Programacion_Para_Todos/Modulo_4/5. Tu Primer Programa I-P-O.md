# Primera Código I-P-O:

Un programa **IPO** (por sus siglas en inglés) es aquel que sigue el modelo fundamental de **Input, Process, Output** (Entrada, Proceso y Salida). Este modelo describe la estructura básica de cómo funciona prácticamente cualquier programa o sistema informático: recibe datos, hace algo con ellos y devuelve un resultado.

```python
#Convertir piso de elevador (Europa - EEUU)
piso_europa = input("Ingrese el piso de elevador en Europa:")
piso_eeuu = int(piso_europa) + 1
print("El piso de elevador en EEUU es:", piso_eeuu)

```

### 1. Entrada (Input)

Es el momento en el que el programa recibe información del exterior (en este caso, del usuario que teclea un número en la consola).

- **En tu código:** **`piso_europa = input("Ingrese el piso de elevador en Europa:")`**
- **Lo que hace:** Detiene la ejecución, espera a que la persona escriba un dato y guarda ese texto ingresado en la variable **`piso_europa`**.

### 2. Proceso (Process)

Es el "cerebro" de la operación. Aquí el programa toma los datos de entrada, los manipula, realiza cálculos o aplica la lógica necesaria para resolver el problema.

- **En tu código:** `piso_eeuu = int(piso_europa) + 1`
- **Lo que hace:** Toma el texto que ingresó el usuario y lo convierte en un número matemático real (`int()`). Una vez convertido, le suma **1** (ya que el piso 0 en Europa suele ser el piso 1 en Estados Unidos). Finalmente, guarda este nuevo valor calculado en la variable `piso_eeuu`.

### 3. Salida (Output)

Es la forma en que el programa entrega el resultado final al usuario o envía esa información para que sea utilizada por otro sistema.

- **En tu código:** `print("El piso de elevador en EEUU es:", piso_eeuu)`
- **Lo que hace:** Imprime en la pantalla el mensaje de texto final acompañado del número que se calculó en el paso anterior, completando así el ciclo del programa.
