# 2.4 … 2.6 Sentencias, operadores y expresiones

## **2.4 Sentencias**

Una **`sentencia`** es una unidad de código que el intérprete de Python puede ejecutar. Hasta ahora hemos visto dos tipos: **`print`** y las asignaciones.

- **En modo interactivo:** el intérprete ejecuta la sentencia y muestra el resultado, si lo hay.
- **En un script:** contiene una secuencia de sentencias que se van ejecutando de una en una.
- Una asignación como **`x = 2`** no produce ninguna salida visual a menos que uses **`print(x)`**.

## **2.5 Operadores y operandos**

Los **operadores** son símbolos especiales que representan cálculos. Los valores a los que se aplican reciben el nombre de **`operandos`**.

- Suma: `+`
- Resta: **`-`**
- Multiplicación: **`*`**
- División: **`/`** (En Python 3, la división normal siempre devuelve un número de punto flotante).
- División entera: **`//`** (Trunca la parte decimal y devuelve un entero).
- Exponenciación: **`**`** (Eleva un número a una potencia).

```python
>>> minute = 59
>>> minute / 60
0.9833333333333333
>>> minute // 60
0
```

## **2.6 Expresiones**

Una **expresión** es una combinación de valores, variables y operadores. Un valor por sí mismo se considera una expresión, al igual que una variable.

Ejemplos de expresiones válidas:

```python
5
x = 5
x + 1

```
