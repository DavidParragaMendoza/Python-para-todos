# **5.1 Actualización de variables**

Un patrón muy común en programación es actualizar el valor de una variable utilizando su valor antiguo.

```python
x = x + 1
```

Esto significa: "toma el valor actual de **`x`**, añádele 1 y luego actualiza **`x`** con el nuevo valor".

**`Regla de oro:`** Antes de poder actualizar una variable, debes **inicializarla** (darle un valor inicial). Si intentas actualizar una variable que no existe, Python dará un **`NameError`**.

- Sumar 1 a una variable se denomina **`incrementar`**.
- Restar 1 se denomina **`decrementar`**.

# **5.2 La sentencia `while`**

Los ordenadores son excelentes para automatizar tareas repetitivas sin cometer errores. Python proporciona la sentencia **`while`** para hacer iteraciones (bucles).

```python
n = 5
while n > 0:
    print(n)
    n = n - 1
print('¡Despegue!')
```

El flujo de ejecución de un bucle **`while`** es:

1. Se evalúa la condición.
2. Si es falsa, se sale del bucle y continúa el programa.
3. Si es verdadera, se ejecuta el cuerpo del **`while`** y luego se vuelve al paso 1.

La variable que cambia cada vez que el bucle se ejecuta y controla cuándo termina recibe el nombre de **`variable de iteración`**.

# **5.3 Bucles infinitos**

Si un bucle carece de una **`variable de iteración`** que cambie para hacer que la condición se vuelva falsa, el bucle se repetirá para siempre. Esto es lo que conocemos como un **`bucle infinito`**.
