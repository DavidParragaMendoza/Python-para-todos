# **5.7 Diseños de bucles**

La mayoría de los bucles se construyen siguiendo este patrón:

1. **`Inicializar`** variables antes del bucle.
2. **`Operar`** con cada elemento en el cuerpo del bucle.
3. **`Revisar`** los resultados al terminar.

## **5.7.1 Bucles de recuento y suma**

- **Para contar:** Usamos una variable inicializada en 0 y le sumamos 1 en cada iteración.
- **Para sumar (acumular):** Usamos una variable inicializada en 0 y le sumamos el valor del elemento en cada iteración. Esta variable recibe el nombre de **`acumulador`**.

```python
total = 0
for valor in[3, 41, 12, 9, 74, 15]:
    total = total + valor
print('Total: ', total)
```

## **5.7.2 Bucles de máximos y mínimos**

Para encontrar el valor mayor o menor, utilizamos la palabra reservada **`None`**, que es una constante especial que indica que una variable está "vacía".

```python
mayor = None
for valor in[3, 41, 12, 9, 74, 15]:
    if mayor is None or valor > mayor:
        mayor = valor
print('Mayor:', mayor)
```

La condición comprueba: "¿Es este el primer valor que veo (`is None`) o es este valor más grande que el mayor valor que he visto hasta ahora?". Si es así, actualizamos la variable `mayor`.

# **5.8 Depuración por Bisección:**

A medida que tu programa crece, buscar errores línea por línea toma demasiado tiempo. Una técnica útil es la **`depuración por bisección`**:

1. Busca un lugar a la mitad de tu código e inserta un **`print()`** para comprobar el valor de las variables.
2. Si los valores allí son correctos, sabes que el problema está en la **segunda mitad** del programa.
3. Si son incorrectos, el problema está en la **primera mitad**.

Al partir el problema por la mitad repetidas veces, reducirás drásticamente la cantidad de líneas de código que debes revisar para encontrar el "bug".
