# **5.4 "Bucles infinitos" y break**

A veces es útil crear un bucle infinito a propósito escribiendo **`while True:`** y utilizar la sentencia **`break`** para salir de él explícitamente cuando se cumpla una condición.

```python
while True:
    linea = input('> ')
    if linea == 'fin':
        break
    print(linea)
print('¡Terminado!')
```

En este patrón, el bucle pide información constantemente. Cuando el usuario escribe "fin", se ejecuta **`break`**, lo que saca al programa del bucle inmediatamente y salta a la sentencia que está después de él.

# **5.5 Finalizar iteraciones con continue**

Si estás dentro de un bucle y quieres saltarte el resto del código **solo para esa iteración actual** y pasar a la siguiente, puedes usar **`continue`**.

```python
while True:
    linea = input('> ')
    if linea[0] == '#' :
        continue
    if linea == 'fin':
        break
    print(linea)
print('¡Terminado!')
```

Si el usuario escribe algo que empieza con **`#`**, el **`continue`** hace que Python ignore el `print(linea)` y vuelva al principio del bucle (al `input`).

# **5.6 Bucles definidos usando `for`**

Mientras que el **`while`** es un bucle indefinido, el bucle **`for`** es un bucle definido: se repite a través de un conjunto conocido de elementos (como una lista de palabras).

```python
amigos =['Joseph', 'Glenn', 'Sally']
for amigo in amigos:
    print('Feliz año nuevo:', amigo)
print('¡Terminado!')
```

En este caso, **`amigo`** es la **variable de iteración**. El bucle se ejecutará exactamente tantas veces como elementos haya en el conjunto (en este caso, tres veces), asignando a **`amigo`** un valor distinto de la lista en cada iteración.
