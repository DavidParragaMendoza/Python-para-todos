# 4.1 … 4.3 Llamadas a funciones, funciones internas y conversión de tipos


# **4.1 Llamadas a funciones**

En programación, una **`función`** es una secuencia de sentencias que realizan una operación y que reciben un nombre. Cuando llamas a una función, especificas su nombre seguido de paréntesis.

```python
>>> type(32)
<class 'int'>
```

En este ejemplo:

- El nombre de la función es **`type`**.
- La expresión entre paréntesis, **`32`**, es el **`argumento`** (el valor o variable que se pasa como entrada).
- El resultado devuelto por la función se conoce como **`valor de retorno`**.

>👨🏻‍🏫  
>Es habitual decir que una función **toma** un argumento y **retorna** (o devuelve) un resultado.


# **4.2 Funciones internas**

Python incluye un conjunto de funciones predefinidas (funciones internas) para resolver problemas comunes sin que tengamos que definirlas nosotros.

Algunos ejemplos útiles con cadenas o listas de valores:

- **`max()`**: Devuelve el valor mayor (el carácter "más grande" alfabéticamente en una cadena).
- **`min()`**: Devuelve el valor menor.
- **`len()`**: Devuelve el número de elementos en su argumento (ej. la cantidad de caracteres en una cadena).

```python
>>> len('Hola, mundo')
11
```

>👨🏻‍🏫  
>Debes tratar los nombres de las funciones internas como si fueran palabras reservadas; evita usar **max** o **min** como nombres de tus variables.


# **4.3 Funciones de conversión de tipos**

Python proporciona funciones internas que convierten valores de un tipo a otro:

- **`int()`**: Toma un valor y lo convierte en entero. Si le pasas un número de punto flotante, corta y descarta la parte decimal (no redondea).
- **`float()`**: Convierte enteros y cadenas en números de punto flotante.
- **`str()`**: Convierte su argumento en una cadena de texto.

```python
>>> int(3.99999)
3
>>> float('3.14159')
3.14159
```
